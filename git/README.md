# Git

Git is a popular version control system.

## Basic workflow

Working directory -> Staging area -> git repository

## Env setup

setup username

`git config --global user.name "berichge"`
`git config --global user.email "berichge@gmail.com"`

list all configs
`git config --list`

## Basic command

git clone
git log
git show `commit sha`
git diff
git status
git add `file`
git push origin master
git pull
